# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to identify factors that influence customer retention and customer loss.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI

## Project Steps
- Data loading and understanding
- Data cleaning
- Exploratory Data Analysis (EDA)
- Customer churn analysis
- Power BI dashboard creation

## Key Insights
- Month-to-month contract customers have higher churn compared to long-term contract customers.
- Customers with shorter tenure are more likely to leave.
- Customer retention can be improved through targeted offers and better onboarding.

## Dashboard
Power BI dashboard includes:
- Total customers
- Churn rate
- Churn by contract
- Churn by internet service
- Tenure analysis
- Monthly charges analysis
